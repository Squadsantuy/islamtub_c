  • 1. Masuk waktu

    • Apabila seseorang bertayamum sebelum masuk waktu salat, maka tayamumnya tidak sah.

  • 2. Niat Bertayamum

    • Apabila seseorang bertayamum akan tetapi tidak niat, maka tayamumnya tidak sah. ( Begitu pula wudhu dan sholat )

  • 3. Beragama Islam

    • Apabila seseorang bertayamum akan tetapi ia tidak beragama islam, maka tayamumnya tidak sah. ( Begitu pula wudhu dan sholat )

  • 4. Berusaha Mencari Air

    • Di usahakan untuk melakukan pencarian terlebih dulu ketika tidak mendapatkan air untuk berwudhu.

  • 5. Tidak Ada Yang Menghalangi Kulit

    • Contonya seperti : seperti lilin, mentega, atau benda lain yang membuat kulit tidak dapat tersentuh secara langsung, Maka hilangkan dulu penghalangnya. ( Begitu pula wudhu )

  • 6. Tidak Dalam Keadaan Haid/Nifas

    • Apabila seseorang bertayamum akan tetapi ia haid/nifas, maka tayamumnya tidak sah. ( Begitu pula wudhu dan sholat )

  • 7. Adanya Udzur ( Halangan )

    • Seseorang diperbolehkan tayamum ketika ia mempunyai Udzur ( halangan ) yang menyebabkan ia tidak boleh berwudhu, Seperti sakit,perang, dan lain-lain.
